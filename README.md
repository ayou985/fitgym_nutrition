# FitGym Nutrition - E-commerce PHP

Plateforme de vente de compléments alimentaires développée avec une architecture **MVC** personnalisée.

##  Fonctionnalités
- **Système d'authentification** sécurisé (Hashage PHP).
- **Gestion de profil** : Modification des infos et upload d'avatar dynamique.
- **Catalogue produits** : Recherche textuelle et filtres par catégorie/prix/saveur.
- **Espace Admin** : CRUD complet pour la gestion des produits et des utilisateurs.
- **Interface responsive** : Optimisée avec Bootstrap 5.

## Stack Technique
- **Backend** : PHP 8 (Architecture MVC)
- **Base de données** : MySQL
- **Frontend** : JavaScript, Bootstrap 5, SweetAlert2
- **Déploiement** : Hébergé sur InfinityFree

## Installation
1. Cloner le dépôt.
2. Importer le fichier `insert_products.sql` dans votre base de données.
3. Configurer le fichier `Config/Database.php`.
4. Lancer avec XAMPP !
